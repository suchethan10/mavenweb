 HOW TO CREATE A MAVEN PROJECT IN ECLIPSE


1. Open Eclipse IDE.

2. Go to:
   File → New → Project

3. In the list, select:
   Maven → Maven Project
   Click Next.

4. On the next screen:
   • DO NOT tick "Create a simple project (skip archetype selection)"
   Click Next.

5. Select an archetype:
   org.apache.maven.archetypes → maven-archetype-quickstart
   Click Next.

6. Fill project details:
   Group Id: com.example
   Artifact Id: MyProject
   Version: (leave default)
   Package: com.example.myproject

   Click Finish.

7. Eclipse will create the Maven structure:
   src/main/java
   src/test/java
   pom.xml

8. To add dependencies:
   Open pom.xml and add inside <dependencies> </dependencies>

   Example:
   <dependency>
       <groupId>javax.servlet</groupId>
       <artifactId>javax.servlet-api</artifactId>
       <version>4.0.1</version>
       <scope>provided</scope>
   </dependency>

9. To run your project:
   Right-click project → Run As → Java Application
